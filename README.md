# DDR4 Pricing Log

Daily tracking of **DDR4 16Gb (2Gx8) 3200** spot prices from [DRAMeXchange](https://www.dramexchange.com).

## Data Format

`ddr4-pricing.json` contains:

| Field | Description |
|-------|-------------|
| `date` | Date of the log entry (YYYY-MM-DD) |
| `daily_high` | Daily high price (USD) |
| `daily_low` | Daily low price (USD) |
| `session_high` | Session high price (USD) |
| `session_low` | Session low price (USD) |
| `session_average` | Session average price (USD) |
| `session_change` | Session change percentage |

## Update Schedule

This log is updated automatically every day at 12:00 AM (GMT+8).

## Source

Data sourced from DRAMeXchange - the world's leading DRAM and NAND Flash market research firm.
